PKBook 静态博客发布程序

PKBook

静态博客发布程序: 使用 PHPKoa 开发，基于 Markdown 文档发布的轻量级博客管理应用。

主要特性

  • 文档管理：采用 Amaze UI 搭建可视化后台管理
  • 文档编辑：采用 editor.md 开源 Markdown 编辑器
  • 用户评论：支持 GitHub Issues 的评论系统
  • 图片上传：支持 qiniu 图片上传
  • 博客主题：支持主题定制
  • 静态发布：后台管理快速发布 html 静态文件
  • 文章搜素：支持全文关键词索引

在线体验

  1. green 使用 Amaze UI 博客主题模板！
  2. next 基于 HEXO 博客主题改写！
  3. gitbook 基于 GitBook 默认主题改写！
